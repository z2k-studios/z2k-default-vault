# Feature 117 — Podcast Interview - Adam Grant

## Description
The Podcast Interview - Adam Grant template is a host-specific preset (Option 3 pattern) for capturing notes from Adam Grant's podcast interviews. It presets the host and show name fields via `{{fieldInfo ... value="..."}}` and includes the shared podcast content structure via the Podcast Interview Content block partial. Source type is set to `.:Z2K/SourceType/Podcast`. Located at `Information/Templates/Podcast Interview - Adam Grant.md`, authored by Z2K Studios, LLC.
