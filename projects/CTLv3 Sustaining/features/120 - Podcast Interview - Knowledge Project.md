# Feature 120 — Podcast Interview - Knowledge Project

## Description
The Podcast Interview - Knowledge Project template is a host-specific preset for capturing notes from The Knowledge Project podcast interviews hosted by Shane Parrish. It presets the host and show name fields via `{{fieldInfo ... value="..."}}` and includes the shared podcast content structure via the Podcast Interview Content block partial. Source type is set to `.:Z2K/SourceType/Podcast`. Located at `Information/Templates/Podcast Interview - Knowledge Project.md`, authored by Z2K Studios, LLC.
