# Feature 114 — Lecture

## Description
The Lecture template provides a structured format for capturing notes from lectures, educational presentations, and course materials. It includes fields for lecturer, institution, topic, and detailed notes organized by session or topic. Located at `Information/Templates/Lecture.md`, authored by Z2K Studios, LLC.
