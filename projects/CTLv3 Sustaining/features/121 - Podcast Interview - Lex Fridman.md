# Feature 121 — Podcast Interview - Lex Fridman

## Description
The Podcast Interview - Lex Fridman template is a host-specific preset for capturing notes from Lex Fridman's podcast interviews. It presets the host and show name fields via `{{fieldInfo ... value="..."}}` and includes the shared podcast content structure via the Podcast Interview Content block partial. Source type is set to `.:Z2K/SourceType/Podcast`. Located at `Information/Templates/Podcast Interview - Lex Fridman.md`, authored by Z2K Studios, LLC.
