# Feature 132 — Information - Details

## Description
The Information - Details block template provides a reusable detailed content section for Information domain document templates. It defines the structure for in-depth notes, extended excerpts, and detailed breakdowns of information sources. Located at `Information/Templates/Information - Details.md`, authored by Z2K Studios, LLC.
