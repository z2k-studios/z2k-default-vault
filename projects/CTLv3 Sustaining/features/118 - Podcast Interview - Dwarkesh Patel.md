# Feature 118 — Podcast Interview - Dwarkesh Patel

## Description
The Podcast Interview - Dwarkesh Patel template is a host-specific preset for capturing notes from Dwarkesh Patel's podcast interviews. It presets the host and show name fields via `{{fieldInfo ... value="..."}}` and includes the shared podcast content structure via the Podcast Interview Content block partial. Source type is set to `.:Z2K/SourceType/Podcast`. Located at `Information/Templates/Podcast Interview - Dwarkesh Patel.md`, authored by Z2K Studios, LLC.
