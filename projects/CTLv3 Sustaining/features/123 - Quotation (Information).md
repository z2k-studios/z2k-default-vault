# Feature 123 — Quotation (Information)

## Description
The Quotation (Information) template provides a structured format for capturing notable quotations within the Information domain. It includes fields for the quote text, attribution, source, and context. Source type is set to `.:Z2K/SourceType/Quotation`. Located at `Information/Templates/Quotation.md`, authored by Z2K Studios, LLC.
