# Feature 125 — Quote an Email

## Description
The Quote an Email template provides a structured format for capturing and referencing content from emails. It includes fields for sender, recipient, date, subject, and quoted content. This template was consolidated from multiple v2 domains into the Information domain. Located at `Information/Templates/Quote an Email.md`, authored by Z2K Studios, LLC.
