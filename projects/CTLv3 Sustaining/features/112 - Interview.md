# Feature 112 — Interview

## Description
The Interview template provides a structured format for capturing notes from generic non-podcast interviews. It covers in-person, written, or video interviews and includes fields for interviewer, interviewee, context, and key discussion points. This template is distinct from the Podcast Interview template which is specifically designed for podcast-format interviews. Located at `Information/Templates/Interview.md`, authored by Z2K Studios, LLC.
