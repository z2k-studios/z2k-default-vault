# Feature 110 — Book

## Description
The Book template provides a structured format for capturing notes, highlights, and metadata from books. It includes fields for author, publisher, reading status, and detailed notes. The template sets the source type to `.:Z2K/SourceType/Book`. Located at `Information/Templates/Book.md`, authored by Z2K Studios, LLC.
