# Feature 127 — Wikipedia Entry

## Description
The Wikipedia Entry template provides a structured format for capturing notes and key information from Wikipedia articles. It includes fields for the article topic, summary, key sections, and references. Located at `Information/Templates/Wikipedia Entry.md`, authored by Z2K Studios, LLC.
