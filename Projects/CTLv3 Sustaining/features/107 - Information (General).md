# Feature 107 — Information (General)

## Description
The Information (General) template is the domain default document template for the Information domain. It provides a general-purpose structure for capturing information notes that don't fit a more specific template. As the domain default, it is used when no other Information template is explicitly selected. Located at `Information/Templates/Information (General).md`, authored by Z2K Studios, LLC.
