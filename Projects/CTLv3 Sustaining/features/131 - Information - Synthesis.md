# Feature 131 — Information - Synthesis

## Description
The Information - Synthesis block template provides a reusable synthesis content section for Information domain document templates. It defines the structure for personal synthesis and interpretation of information sources. This block uses the `[!me]` callout format to distinguish personal synthesis from source material. Located at `Information/Templates/Information - Synthesis.md`, authored by Z2K Studios, LLC.
