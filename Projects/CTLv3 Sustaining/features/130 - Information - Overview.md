# Feature 130 — Information - Overview

## Description
The Information - Overview block template provides a reusable overview content section for Information domain document templates. It defines the structure for high-level overviews of information sources, providing broader context than the summary block. Located at `Information/Templates/Information - Overview.md`, authored by Z2K Studios, LLC.
