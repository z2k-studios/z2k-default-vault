# Feature 119 — Podcast Interview - Huberman Lab

## Description
The Podcast Interview - Huberman Lab template is a host-specific preset for capturing notes from Huberman Lab podcast interviews. It presets the host and show name fields via `{{fieldInfo ... value="..."}}` and includes the shared podcast content structure via the Podcast Interview Content block partial. Source type is set to `.:Z2K/SourceType/Podcast`. Located at `Information/Templates/Podcast Interview - Huberman Lab.md`, authored by Z2K Studios, LLC.
