# Feature 128 — Podcast Interview Content

## Description
The Podcast Interview Content block template provides the core shared content structure used by all podcast interview templates. It is included via `{{> "Podcast Interview Content"}}` by both the generic Podcast Interview template (116) and all host-specific podcast presets (117-122). This block defines the common podcast note-taking structure including fields for episode details, guest information, topics discussed, and key takeaways. Located at `Information/Templates/Podcast Interview Content.md`, authored by Z2K Studios, LLC.
