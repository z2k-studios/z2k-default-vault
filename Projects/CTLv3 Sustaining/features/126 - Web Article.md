# Feature 126 — Web Article

## Description
The Web Article template provides a structured format for capturing notes from web articles and online content. It includes fields for URL, author, publication date, and key content. Source type is set to `.:Z2K/SourceType/WebArticle`. Located at `Information/Templates/Web Article.md`, authored by Z2K Studios, LLC.
